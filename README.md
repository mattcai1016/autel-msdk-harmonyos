# MSDK 平台示例

本目录包含展示 MSDK 在不同平台上使用的示例应用程序。每个平台特定的目录都包含完整的、可运行的示例和详细文档。

## 目录结构

```
examples/
├── README.md           # 本文件
├── Harmony/           # HarmonyOS 平台示例
│   ├── README.md      # 平台特定文档
│   └── [examples]/    # 独立示例应用程序
├── Linux/             # Linux 平台示例
│   ├── README.md      # 平台特定文档
│   └── [examples]/    # 独立示例应用程序
└── [其他平台]/        # 其他平台示例
```

## 平台特定示例

### HarmonyOS
`Harmony/` 目录包含专门为 HarmonyOS 平台设计的示例。每个示例都展示了 MSDK 的不同集成和使用方面。

### Linux
`Linux/` 目录包含适用于 Linux 系统的示例。这些示例涵盖了 MSDK 的各种特性和使用场景。

## 示例结构

每个示例应用程序都遵循以下标准结构：

```
[示例名称]/
├── README.md         # 示例特定文档
├── src/              # 源代码
├── include/          # 头文件
├── CMakeLists.txt    # 构建配置
└── docs/             # 附加文档
```

## 文档指南

每个示例都应包含：

1. **概述**：简要说明示例演示的内容
2. **前提条件**：所需的依赖项和设置
3. **构建说明**：分步构建过程
4. **使用方法**：如何运行和使用示例
5. **代码说明**：关键实现细节
6. **预期输出**：示例输出或行为
7. **故障排除**：常见问题和解决方案

## 贡献指南

添加新示例时：

1. 在相应平台文件夹下创建新目录
2. 遵循标准示例结构
3. 包含完整的文档
4. 确保示例完整且可运行
5. 在目标平台上测试示例
6. 如果添加新平台，更新本 README.md

## 支持

对于示例相关的问题或疑问，请：

1. 查看平台特定的 README.md
2. 查看示例的文档
3. 在项目仓库中提交 issue
